# ejecutar en desarrollo

1. clonar el repositorio
2. ejecutar el siguiente comando para instalar las dependencias

```
yarn intall
```
3. tener nest cli
```
yarn global add @nestjs/
```

4. levantar la base de datos
```
docker-compose up -d
```
